# 🎵 AI Music Generator

An AI-powered music generator built using TensorFlow and MIDI datasets.

## Features
- Train on MIDI datasets
- Generate original melodies
- TensorFlow-based neural network
- MIDI file processing using music21

## Tech Stack
- Python
- TensorFlow
- music21
- NumPy
- Git & GitHub

## How to Run

### Train the Model
```bash
python Train.py
```

### Generate Music
```bash
python Generate.py
```
